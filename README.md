# focusnflow

**demo video:** https://youtu.be/HamKvCo2O_A

flutter app that helps college students plan their study time, find open rooms on campus, and study with friends. built with firebase for real-time sync.

## features

- task planner with due dates, effort hours, and course weight
- auto-generated weekly study schedule based on task urgency
- real-time study room finder with occupancy tracking
- study groups with group chat
- shared pomodoro timer for group sessions
- firebase auth for login/register
- swipe to delete tasks

## how to run

1. clone the repo
2. run `flutter pub get`
3. set up firebase with `flutterfire configure`
4. run `flutter run`

## tech stack

- flutter
- firebase auth
- cloud firestore
- firebase storage
- firebase cloud messaging
